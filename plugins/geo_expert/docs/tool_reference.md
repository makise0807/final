# Geo Expert Tool Reference

This document describes the tools provided by the Geo Expert Hermes Pack.

The tools are intended to be registered in Hermes toolsets and used through the Hermes mainline agent loop.

## Tool List

The package provides six high-level tools:

```text
geo_expert.run_preliminary_case_check
geo_expert.search_sop_database
geo_expert.search_legal_database
geo_expert.preview_satellite_overlay
geo_expert.open_last_outputs
geo_expert.handle_approval
